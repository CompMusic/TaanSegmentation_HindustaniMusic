Main Code that needs to be run: TaanSegmentation.m

Input: Text file with tab separated values with 2 columns. First column with Time and second with extracted melody pitch value in Hz
The melody should be extracted from the concert using Melodia at its default settings.

Output: Array with StartTime followed by EndTime of Taan segments in a concert.
For example: StartTime1 EndTime1 StartTime2 EndTime2....... StartTimeN EndTimeN
